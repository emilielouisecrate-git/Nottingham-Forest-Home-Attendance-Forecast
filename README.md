# Nottingham Forest Home Attendance Forecast

Time series analysis of Nottingham Forest's 2024-2025 Premier League home attendance using Meta's Prophet forecasting framework. 

## Overview
This project explores the behaviour of match attendance at The City Ground during the 2024-2025 Premier League season. A time series approach is used to analyse trends and produce a short-term forecast of attendance levels. 

## Methods
- Time series visualisation
- Forecasting using Meta Prophet
- Model interpretation and limitations analysis
